# Roadmap

1. Pre-built test arena, e.g. nagrand
2. One class, one level
3. Target dummy, character always passes turn
4. Character can move
5. Character can make melee attack

# Design Doc (sorta)

* PF2E Ruleset [1,2]
* Hex Grid
  * 5ft diameter

* Randomly generated maps/arenas based on config options
* Randomly generated enemies based on config options
    * unique and minion packs, champion packs
        * special properties, gear, name, etc.
* Randomly modified super-unique enemies
    * base enemies with some random modification
* Boss enemies
    * special enemies with consistently defined stats
    * maybe random gearing?
